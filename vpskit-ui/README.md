# VPSKit UI

Minimal React and TypeScript source skeleton for the VPSKit control panel.

## Included

- A small status dashboard in `src/App.tsx`
- React entrypoint in `src/main.tsx`
- Shared styling in `src/styles.css`

## Local development

```bash
npm install
npm run dev
```

The UI currently uses local sample data only. API wiring should be added after backend routes stabilize.

# VPSKit v1.0 Safety Boundary

## Core

The stable core is infra only.

- no money movement
- no automated fulfillment
- no hidden background actions

## SaaS

The SaaS layer is mock-only.

- static order states
- simulated bot replies
- no network side effects

## Client

The client layer is static content only.

- landing pages
- docs site
- release links

## Rule

No runtime coupling from core into SaaS or client layers.

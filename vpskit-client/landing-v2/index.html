VPSKit v1.0

Stable infra toolkit for fast release packaging.

VPSKit keeps the operational layer tight: QA, verify, bundle, demo, and release packaging in one clean path. The public site stays focused on clarity, speed, and conversion.

QA

Fast read-only checks that keep the release path honest.

Verify

Protocol checks for the stable infra surface.

Bundle

Redacted-safe client packaging for release handoff.

$ vpskit qa --redact
VPSKIT_QA=pass
VERIFY_VLESS_REALITY=pass
VERIFY_HYSTERIA2=pass
VERIFY_TROJAN=pass
$ vpskit sub bundle --redact
SUB_BUNDLE=pass
SENSITIVE_OUTPUT=redacted
$ vpskit demo package
DEMO_PACKAGE=ready

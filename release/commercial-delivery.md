## What to Ask the Customer

- VPS provider and location
- SSH access method
- Domain or DNS ownership status
- Desired protocol mix
- Whether Shadowrocket is the target client
- Current TCP 443 status

## Delivery Workflow

1. Collect customer details.
2. Run `vpskit doctor`.
3. Repair subscription if needed.
4. Deliver install and troubleshooting docs.
5. Confirm import on the target client.

## Handoff Checklist

- Subscription file verified
- DNS checked
- TCP probe checked
- Repair report shared
- Support window confirmed

## Support Checklist

- Re-run `doctor`
- Re-run `sub`
- Use local repair only
- Avoid system mutation outside approved phases

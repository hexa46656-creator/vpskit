# VPSKit v0.7.0-beta Client Bundle Checklist

## Preflight

- Confirm the branch is `v0.7.0-beta-unified-client-bundle`
- Confirm the base includes `v0.6.3-beta` or `173dd91` or newer
- Confirm the install is complete and the subscription files exist

## Bundle Generation

```bash
vpskit sub bundle --redact --output ./vpskit-client-bundle
vpskit sub bundle --redact --force --output ./vpskit-client-bundle
```

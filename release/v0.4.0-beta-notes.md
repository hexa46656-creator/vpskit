# VPSKit v0.4.0-beta Notes

## Status

`v0.4.0-beta` keeps the Phase 1 VLESS Reality baseline and adds subscription export compatibility for existing client formats.

## Included

- `vpskit sub show`
- `vpskit sub formats`
- `vpskit sub export raw`
- `vpskit sub export base64`
- `vpskit sub export shadowrocket`
- `vpskit sub export v2rayng`
- `vpskit sub export clash-meta`
- `vpskit sub export sing-box`
- Existing VLESS Reality URI parsing from the current subscription file
- Minimal Clash Meta YAML export for the existing VLESS Reality URI
- Minimal sing-box JSON export for the existing VLESS Reality URI

## Non-Goals

- Hysteria2
- Trojan
- SaaS control plane
- Telegram Bot
- PayPal or billing automation
- Web UI
- QR code generation

## Known Limitations

- This release only exports the existing VLESS Reality config that was already installed.
- It does not change the installed Xray config format.
- Missing or malformed subscription files fail closed with an error message.
- QR code generation is not included yet.

## Operator Guidance

Use the raw URI for clients that accept the existing VLESS Reality link directly, or choose a format-specific export when importing into Clash Meta or sing-box.

Recommended commands:

```bash
bash vpskit/cli/vpskit.sh sub show
bash vpskit/cli/vpskit.sh sub formats
bash vpskit/cli/vpskit.sh sub export shadowrocket
bash vpskit/cli/vpskit.sh sub export v2rayng
bash vpskit/cli/vpskit.sh sub export clash-meta
bash vpskit/cli/vpskit.sh sub export sing-box
bash vpskit/cli/vpskit.sh sub export base64
```

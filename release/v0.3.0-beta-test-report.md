## Known Limitations

- The official Xray installer uses `User=nobody`, which can produce a systemd warning on some hosts. This is accepted for the current beta.
- Rollback does not fully reverse apt installs, Linux user creation, sudo group changes, UFW state changes, or service restarts.
- This report does not add or validate Hysteria2, Trojan, SaaS, Telegram Bot, PayPal, or Web UI functionality.
